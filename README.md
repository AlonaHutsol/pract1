# Практична робота №1

## Тема
MS Visual Studio. Графіка в C++. Анімація. Звук.

## Мета
Навчитися створювати графічні примітиви в середовищі MS Visual Studio за допомогою мови програмування C++.

## Завдання
1. Ознайомитися з можливостями середовища розробки MS Visual Studio для створення графічних програм.
2. Навчитися використовувати графічні примітиви для побудови різних фігур.
3. Реалізувати малювання графічних об’єктів (лінія, прямокутник, еліпс, трикутник, зірка, сектор кола, будинок).
4. Додати можливість вибору кольору фігури за допомогою елемента ComboBox.
5. Реалізувати анімацію з використанням компонентів Timer та PictureBox.
6. Додати відтворення звуку за допомогою звукових файлів формату `.wav`.
7. Використати клас `SoundPlayer` для завантаження та програвання звукових файлів.
